---
name: vetiver
type: note
aliases: [haitian vetiver, java vetiver, vetiver root]
family: woody
---

Earthy, smoky, slightly sweaty in the best possible way. Smells like damp roots after rain. Haitian vetiver is the cleanest and most citrus-tinged; Java is darker and more leathery. The note that gives a fragrance "skin" — close, lived-in, not perfumey.
